FAQ

How Much Does Roboweb Cost?

Roboweb is currently free but you pay for your usage of OpenAPI via the use of your OpenAI API key.

Roboweb currently uses gpt-3.5-turbo. Refer to the OpenAI pricing page for the latest pricing.

How Is My OpenAI API Key Used?

Your OpenAI API Key is used to make calls to OpenAI on your behalf. Your API key is stored in your browser’s local storage and is never transmitted to Roboweb’s servers.

What Data Is Stored On Roboweb’s Servers?

Your chat history is stored on Roboweb’s servers in order to persist your chat history across sessions.

How can I get help?

Email us here roboweb-users@googlegroups.com.

Where can I file a bug or feature request?

You can open a GitHub Issue in jlewi/roboweb-docs.
