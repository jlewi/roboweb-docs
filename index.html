Quickstart

Install the JupyterLab Extension Install the extensions python -m pip install roboweb-extension python -m pip install roboweb-server-extension Activate the extension python -m jupyter serverextension enable --py roboweb_server_extension Configure the Extension To use the extension start jupyter lab. jupyter-lab In the right hand navigation window click the roboweb icon. This will open up the roboweb extension in the right half of the screen. The Roboweb assistant should appear in the right hand side toolbar in JupyterLab.
