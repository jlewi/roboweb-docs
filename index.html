Roboweb

    Exploratory Programming With AI

    Roboweb is an AI assistant for exploratory programming. It embeds OpenAI’s ChatGPT in JupyterLab to create an ideal environment for exploratory programming.

    Here’s a short video demo.

    To get started choose one of the following

    • Quickstart to install the extension inside your JupyterLab environment.
    • Kubernetes to use our Docker image with JupyterLab and the extension preinstalled.
